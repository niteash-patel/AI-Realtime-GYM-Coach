# AI Gym Coach

## Overview

AI Gym Coach is a computer vision-based fitness assistant that helps users perform exercises with proper form. Using real-time pose estimation, the system tracks body movements, counts repetitions, and provides feedback to improve workout performance.

## Features

- Real-time pose detection
- Exercise repetition counting
- Form correction and posture analysis
- Live webcam support
- Performance tracking
- AI-powered fitness assistance

## Supported Exercises

- Push-ups
- Squats
- Bicep Curls
- Lunges
- Planks

## Technologies Used

- Python
- OpenCV
- MediaPipe
- NumPy
- Streamlit / Flask
- Machine Learning

## How It Works

1. Start the application
2. Select an exercise
3. Enable webcam access
4. Perform the exercise
5. View real-time rep count and posture feedback

## Installation

```bash
git clone https://github.com/your-username/ai-gym-coach.git
cd ai-gym-coach
pip install -r requirements.txt
```

## Run the Application

```bash
python main.py
```

or

```bash
streamlit run main.py
```
